Keyword Optimization Rules

🛑 Automated Pausing Logic (OR-based):

  1. Performance Threshold: ACOS > configured limit (default: 20%)
  2. Conversion Efficiency: clicks ≥ max threshold (default: 50) AND orders = 0

✅ Active Keyword Criteria:

  • ACOS ≤ configured threshold AND conversion history exists
  • Performance trend analysis indicates positive ROI trajectory

Product Optimization Matrix

🛑 Product Pausing Logic:

  1. Standard Products: Actual ACOS > product threshold (with sales > 0)
  2. Zero-Sales Products: Hypothetical ACOS > product threshold (database-calculated)
  3. Hybrid Analysis: Combined actual + predictive performance scoring

Advanced Placement Bid Engine

🎯 Standard Optimization Algorithm:

  • Base CPC Calculation: minimum_rpc × target_acos (with safety bounds)
  • Adjustment Formula: (placement_rpc ÷ min_rpc - 1) × 100 = bid_adjustment_%
  • Intelligent Scaling: Automatic base CPC multiplication when adjustments exceed 900%
  • Zero-Sales Handling: Temporary €1 sales value for meaningful RPC calculations

⚡ Low-Traffic Special Rules:

  • Trigger: Top-of-Search placement with <20 clicks
  • Action: +100% Top-of-Search adjustment (capped at €1.50 max bid)
  • Other Placements: Maintained at current levels
  • Bid Protection: Automatic scaling to respect Amazon's €1.50 maximum

Revenue Per Click (RPC) Analytics

🔢 Advanced RPC Calculation:
  • Formula: sales ÷ clicks (with null safety)
  • Adjusted Sales: Zero-sales scenarios use €1 for meaningful calculations
  • Minimum RPC: Lowest RPC across all campaign placements
  • Scaling Logic: Base CPC multiplied by integer factor when needed
